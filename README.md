## Setup guide

1. install stow 
```bash
sudo pacman -S stow
```
2. clone repo in $HOME
```bash
cd
git clone https://github.com/martinellegard/dotfiles-stow.git
```
3. setup dots
```bash
cd dotfiles-stow
stow .
```
